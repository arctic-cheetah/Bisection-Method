## Re

```
Re(z)
```

> returns the real component of the complex number `z`.
 
See
* [Wikipedia - Complex number](https://en.wikipedia.org/wiki/Complex_number)

### Examples

```
>> Re(3+4I)
3

>> Im(0.5 + 2.3*I)
2.3
```
