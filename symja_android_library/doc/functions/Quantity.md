## Quantity

```
Quantity(value, unit)
```

> returns the quantity for `value` and `unit`

See 
* [Wikipedia - International System of Units](https://en.wikipedia.org/wiki/International_System_of_Units)

### Examples
 
```
>> Quantity(50, "s") + Quantity(1, "min")
110[s]
```
