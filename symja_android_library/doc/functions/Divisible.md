## Divisible

```
Divisible(n, m)
```

> returns `True` if `n` could be divide by `m`. 

See:  
* [Wikipedia - Divisor](https://en.wikipedia.org/wiki/Divisor)


### Examples  

```
>> Divisible(42,7)
True
```
