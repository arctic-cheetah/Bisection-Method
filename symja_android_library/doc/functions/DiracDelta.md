## DiracDelta

```
DiracDelta(x)
```

> `DiracDelta` function returns `0` for all real numbers `x` where `x != 0`.
 

### Examples
```
>> DiracDelta(-42)
0
```

`DiracDelta` doesn't evaluate for `0`:

```
>> DiracDelta(0)
DiracDelta(0)
```
