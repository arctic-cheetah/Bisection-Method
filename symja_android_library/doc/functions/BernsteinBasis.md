## BernsteinBasis

```
BernsteinBasis(n, v, expr)
```

> computes the Bernstein basis for the expression `expr`.

See:  
* [Wikipedia - Bernstein polynomial](https://en.wikipedia.org/wiki/Bernstein_polynomial) 


### Examples

```
>> BernsteinBasis(3,2,0.3) 
0.189
```
