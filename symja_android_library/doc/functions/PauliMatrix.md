## PauliMatrix

``` 
PauliMatrix(n)
```

> returns the Pauli `2x2` matrix for `n` between `0` and `4`.

See
* [Wikipedia - Pauli matrices](https://en.wikipedia.org/wiki/Pauli_matrices) 

### Examples

```
>> PauliMatrix(2) 
{{0,-I},{I,0}}
```
