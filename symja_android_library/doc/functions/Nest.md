## Nest

```
Nest(f, expr, n)
```
> starting with `expr`, iteratively applies `f` `n` times and returns the final result.

### Examples
 
```
>> Nest(f, x, 3)
f(f(f(x)))
 
>> Nest((1+#) ^ 2 &, x, 2)
(1+(1+x)^2)^2
```
