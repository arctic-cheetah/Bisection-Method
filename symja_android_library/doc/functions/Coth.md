## Coth

```
Coth(z)
```

> returns the hyperbolic cotangent of `z`.
  
See
* [Wikipedia - Hyperbolic function](https://en.wikipedia.org/wiki/Hyperbolic_function)

### Examples
```
>> Coth(0)  
ComplexInfinity
```
