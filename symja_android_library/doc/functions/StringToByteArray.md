## StringToByteArray
 
```
StringToByteArray(string)
```
 
> encodes the `string` into a sequence of bytes using the default character set `UTF-8`, storing the result into into a `ByteArray`. 

See:    
* [Wikipedia - Base64](https://en.wikipedia.org/wiki/Base64)
