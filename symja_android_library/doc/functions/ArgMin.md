## ArgMin

```
ArgMin(function, variable)
```

> returns a minimizer point for a univariate `function`.

See:
* [Wikipedia - Arg max](https://en.wikipedia.org/wiki/Arg_max)

### Examples

```
>> ArgMin(x*10+x^2, x)
-5
```
