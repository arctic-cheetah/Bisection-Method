## PartitionsQ

```
PartitionsQ(n)
```
> gives the number of partitions of the integer `n` into distinct parts

See
* [Wikipedia - Partition (number theory)](https://en.wikipedia.org/wiki/Partition_(number_theory))

### Examples

```
>> PartitionsQ(50)
3658

>> PartitionsQ(6)
4

```
