## PermutationCyclesQ

```
PermutationCyclesQ(cycles-expression)
```

> if `cycles-expression` is a valid `Cycles({{...},{...}, ...})` expression return `True`.
 
See 
* [Wikipedia - Permutation](https://en.wikipedia.org/wiki/Permutation)
	 
### Examples

```
>> PermutationCyclesQ(Cycles({{1, 6, 2}, {4, 11, 12, 3}}))
True
```
