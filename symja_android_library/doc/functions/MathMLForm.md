## MathMLForm

```
MathMLForm(expr)
```

> returns the MathML form of the evaluated `expr`. 
 
See
* [Wikipedia - MathML](https://en.wikipedia.org/wiki/MathML)

### Examples
 
```
>> MathMLForm(D(Sin(x)*Cos(x),x))
```
