## ArgMax

```
ArgMax(function, variable)
```

> returns a maximizer point for a univariate `function`. 

See:
* [Wikipedia - Arg max](https://en.wikipedia.org/wiki/Arg_max)
 
### Examples

```
>> ArgMax(x*10-x^2, x) 
5
```
