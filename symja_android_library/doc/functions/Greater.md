## Greater

```
Greater(x, y) 

x > y
```

> yields `True` if `x` is known to be greater than `y`.

```
lhs > rhs
```

> represents the inequality `lhs > rhs`.
 
	
### Examples
 
```
>> Pi>0
True

>> {Greater(), Greater(x), Greater(1)}
{True, True, True}
```
