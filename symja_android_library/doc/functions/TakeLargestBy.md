## TakeLargestBy

```
TakeLargestBy({e_1, e_2, ..., e_i}, function, n) 
```

> returns the `n` values from the list `{e_1, e_2, ..., e_i}`, where `function(e_i)` is largest.

### Examples

```
>> TakeLargestBy(Prime(Range(10)),Mod(#,7)&, 3) 
{13,5,19}
```
