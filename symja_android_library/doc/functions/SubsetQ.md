## SubsetQ

```
SubsetQ(set1, set2)
```

> returns `True` if `set2` is a subset of `set1`.
	 
See:  
* [Wikipedia - Subset](https://en.wikipedia.org/wiki/Subset)

### Examples

```
>> SubsetQ({a,b,c},{a,b})
True
```
