## HurwitzZeta

```
HurwitzZeta(s, a)
```

> returns the Hurwitz zeta function.

See: 
* [Wikipedia - Hurwitz zeta function](https://en.wikipedia.org/wiki/Hurwitz_zeta_function)
* [Fungrim - Hurwitz zeta function](http://fungrim.org/topic/Hurwitz_zeta_function/)

### Examples

```
>> HurwitzZeta(3,1/2) 
7*Zeta(3)
```
