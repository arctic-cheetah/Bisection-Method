## SlotSequence

```
##
```

> is the sequence of arguments supplied to a pure function.

```
##n
```

> starts with the `n`-th argument. 
 
 
### Examples

``` 
>> Plus(##)& [1, 2, 3]
5

>> Plus(##2)& [1, 2, 3]
5

>> ## // InputForm
##1
```
