## MeanFilter

```
MeanFilter(list, r)
```

> filter which evaluates the `Mean` of `list` for the radius `r`. 
 

### Examples

```
>> MeanFilter({-3, 3, 6, 0, 0, 3, -3, -9}, 2) 
{2,3/2,6/5,12/5,6/5,-9/5,-9/4,-3} 
```
