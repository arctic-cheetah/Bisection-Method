## TakeLargest

```
TakeLargest({e_1, e_2, ..., e_i}, n) 
```

> returns the `n` largest real values from the list `{e_1, e_2, ..., e_i}`.

### Examples

```
>> TakeLargest(Prime(Range(10)), 3) 
{29,23,19}
```
