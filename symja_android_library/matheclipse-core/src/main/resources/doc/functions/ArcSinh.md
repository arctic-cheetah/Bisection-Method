## ArcSinh

```
ArcSinh(z)
```

> returns the inverse hyperbolic sine of `z`.

See:
* [Wikipedia: Inverse hyperbolic function](https://en.wikipedia.org/wiki/Inverse_hyperbolic_function)

### Examples

```
>> ArcSinh(0)
0

>> ArcSinh(0.0)
0.0

>> ArcSinh(1.0)
0.8813735870195429
```
