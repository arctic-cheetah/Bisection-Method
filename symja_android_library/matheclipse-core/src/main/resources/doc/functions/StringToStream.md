## StringToStream

```
StringToStream("string")
```

> converts a `string` to an open input stream.

### Examples

``` 
>> str = StringToStream("4711 dummy 0815")
InputStream[Name: String Unique-ID: 1]

>> Read(str, Number) 
4711

>> Read(str, {Word, Number})
{dummy,0815}

>>Close(str) 
String 
```
