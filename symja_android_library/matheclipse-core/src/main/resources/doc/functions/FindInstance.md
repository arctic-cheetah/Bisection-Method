## FindInstance 

```
FindInstance(equations, vars)
```

> attempts to find one solution which solves the `equations` for the variables `vars`.

### Examples

```
>> FindInstance({x^2==4,x+y^2==6}, {x,y})
{{x->-2,y->-2*Sqrt(2)}}
```
