## DeleteDuplicates

```
DeleteDuplicates(list)
```

> deletes duplicates from `list`. 
 
 
### Examples 
 
```
>> DeleteDuplicates({1, 7, 8, 4, 3, 4, 1, 9, 9, 2, 1})
{1,7,8,4,3,9,2} 
```

```
>> DeleteDuplicates({3,2,1,2,3,4}, Less)
{3,2,1}
```
