## Sign

```
Sign(x)
```

> gives `-1`, `0` or `1` depending on whether `x` is negative, zero or positive.
 
See
* [Wikipedia - Sign function](https://en.wikipedia.org/wiki/Sign_function)

### Examples

```
>> Sign(-2.5)
-1
```
