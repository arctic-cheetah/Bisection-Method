## RightComposition

```
RightComposition(sym1, sym2,...)[arg1, arg2,...]
```

> creates a composition of the symbols applied in reversed order at the arguments.


### Examples

```
>> RightComposition(u, v, w)[x, y]
w(v(u(x,y)))
```
