## StringContainsQ

```
StringContainsQ(str1, str2)
```

> return a list of matches for `"p1", "p2",...` list of strings in the string `str`.
  
### Examples

```
>> StringContainsQ({"the quick brown fox", "jumps", "over the lazy dog"}, "the")
{True,False,True}
```
