## HarmonicNumber

```
HarmonicNumber(n)
```

> returns the `n`th harmonic number.  
 
See
* [Wikipedia - Harmonic number](https://en.wikipedia.org/wiki/Harmonic_number)
 
### Examples

```
>> Table(HarmonicNumber(n), {n, 8})
{1,3/2,11/6,25/12,137/60,49/20,363/140,761/280} 
```
