## TrueQ

```
TrueQ(expr) 
```

> returns `True` if and only if `expr` is `True`.

See:  
* [Wikipedia - Truth value](http://en.wikipedia.org/wiki/Truth_value)

### Examples
 
```
>> TrueQ(True)
True

>> TrueQ(False)
False

>> TrueQ(a)
False
```
