## Div 

```
Div({f1, f2, f3,...},{x1, x2, x3,...})
```

> compute the divergence.

See:  
* [Wikipedia - Divergence](http://en.wikipedia.org/wiki/Divergence)
 
### Examples

```
>> Div({x^2, y^3},{x, y})
2*x+3*y^2
```
