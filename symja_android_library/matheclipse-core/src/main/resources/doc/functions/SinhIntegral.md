## SinhIntegral

```
SinhIntegral(expr)
```

> returns the sine integral of `expr`.
  
See
* [Wikipedia - Trigonometric integral](https://en.wikipedia.org/wiki/Trigonometric_integral)

### Examples

```
>> SinhIntegral(0)
0 
```
