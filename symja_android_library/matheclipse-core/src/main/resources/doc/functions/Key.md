## Key

```
Key(key) 
```

> represents a `key` used to access a value in an association. 
 
```
association[[Key(key)]]
```
 
> determine the value associated to `key` in `association`.

### Examples

```
>> <|1 -> a, 3 -> b|>[[Key(3)]]
b
```
