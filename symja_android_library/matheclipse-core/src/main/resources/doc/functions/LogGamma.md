## LogGamma

```
LogGamma(z) 
```

> is the logarithmic gamma function on the complex number `z`.


See
* [Wikipedia - Gamma function](https://en.wikipedia.org/wiki/Gamma_function) 
* [Fungrim - Gamma function](http://fungrim.org/topic/Gamma_function/)

### Examples

```
>> LogGamma(1/2)
Log(Pi)/2
 
```
