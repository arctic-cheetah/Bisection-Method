## TeXForm

```
TeXForm(expr)
```

> returns the TeX form of the evaluated `expr`. 
 
See
* [Wikipedia - LaTeX](https://en.wikipedia.org/wiki/LaTeX)

### Examples

```
>> TeXForm(D(sin(x)*cos(x),x))
"{\cos(x)}^{2}-{\sin(x)}^{2}"
```
