## BarChart 

```
BarChart(list-of-values, options)
```

> plot a bar chart for a `list-of-values` with option `BarOrigin->Bottom` or `BarOrigin->Bottom`

### Examples
 
```
>> BarChart({1, -2, 3}, BarOrigin->Bottom)
```
