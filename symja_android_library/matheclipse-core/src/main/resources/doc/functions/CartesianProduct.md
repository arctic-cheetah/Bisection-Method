## CartesianProduct

```
CartesianProduct(list1, list2)
```

> returns the cartesian product for multiple lists.

See:  
* [Wikipedia - Cartesian product](http://en.wikipedia.org/wiki/Cartesian_product)  

### Examples

```
>> CartesianProduct({1,2},{3,4})
{{1,3},{1,4},{2,3},{2,4}}
```
