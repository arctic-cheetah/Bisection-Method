## TrigToExp

```
TrigToExp(expr)
```

> converts trigonometric functions in `expr` to exponentials.

See
* [Wikipedia - Euler's formula)](https://en.wikipedia.org/wiki/Euler%27s_formula)  

### Examples

```
>> TrigToExp(Cos(x))
1/2*E^(I*x)+1/2*E^(-I*x)
```
