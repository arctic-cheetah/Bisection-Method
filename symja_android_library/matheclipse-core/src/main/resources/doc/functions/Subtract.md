## Subtract

```
Subtract(a, b)

a - b
```

> represents the subtraction of `b` from `a`. 
 

### Examples

```
>> 5 - 3
2

>> a - b // FullForm
"Plus(a, Times(-1, b))"

>> a - b - c
a-b-c

>> a - (b - c)
a-b+c
```
