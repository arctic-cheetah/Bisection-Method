## EulerPhi

```
EulerPhi(n)
```

> compute Euler's totient function.

See:
* [Wikipedia - Euler's totient function](http://en.wikipedia.org/wiki/Euler%27s_totient_function)
* [Fungrim - Totient function](http://fungrim.org/topic/Totient_function/)

### Examples

```
>> EulerPhi(10)
4
```
