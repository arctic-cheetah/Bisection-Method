## Intersection

```
Intersection(set1, set2, ...)
```

> get the intersection set from `set1` and `set2` ....

See
* [Wikipedia - Intersection (set theory)](https://en.wikipedia.org/wiki/Intersection_(set_theory)) 

### Examples

```
>> Intersection({a,a,b,c},{b,a})
{a,b}
```
