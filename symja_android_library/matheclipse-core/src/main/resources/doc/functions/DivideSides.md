## DivideSides

```
DivideSides(compare-expr, value) 
```

> divides all elements of the `compare-expr` by `value`. `compare-expr` can be `True`, `False` or a comparison expression with head `Equal, Unequal, Less, LessEqual, Greater, GreaterEqual`. 

### Examples

```
>> DivideSides(a==a, x) 
True      
 
>> DivideSides(a==b, x)    
a/x==b/x
```
