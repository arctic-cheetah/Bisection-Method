## Unitize

```
Unitize(expr)
```

> maps a non-zero `expr` to `1`, and a zero `expr` to `0`. 
 
### Examples

```
>> Unitize((E + Pi)^2 - E^2 - Pi^2 - 2*E*Pi)
0
```
