## Factorial2

```
Factorial2(n)

n!!
```

> returns the double factorial number of the integer `n`.
 
See
* [Wikipedia - Double Factorial](https://en.wikipedia.org/wiki/Factorial#Double_factorial)

### Examples

```
>> Factorial2(3)
3
```
