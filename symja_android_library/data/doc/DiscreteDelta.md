## DiscreteDelta

```
DiscreteDelta(n1, n2, n3, ...)
```

> `DiscreteDelta` function returns `1` if all the `ni` are `0`. Returns `0` otherwise.
 

### Examples
``` 
>> DiscreteDelta(0, 0, 0.0)
1
```
