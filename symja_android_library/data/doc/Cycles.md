## Cycles

```
Cycles(a, b)
```

> expression for defining canonical cycles of a permutation.

See:
* [Wikipedia: Cyclic permutation](https://en.wikipedia.org/wiki/Cyclic_permutation)

### Examples

The singletons `{2}` and `{5}` are deleted:

```
>> PermutationCycles({4,2,7,6,5,8,1,3}) 
Cycles({{1,4,6,8,3,7}})
```
