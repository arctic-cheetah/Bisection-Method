## HarmonicMean

```
HarmonicMean({a, b, c,...})
```

> returns the harmonic mean of `{a, b, c,...}`. 
   

See:
* [Wikipedia - Harmonic mean)](https://en.wikipedia.org/wiki/Harmonic_mean)


### Examples

```
>> HarmonicMean({1, 2, 3, 4,5, 6, 7})
980/363
```
