## Haversine

```
Haversine(z)
```

> returns the haversine function of `z`.
 
See:
* [Wikipedia - Haversine formula](https://en.wikipedia.org/wiki/Haversine_formula)

### Examples
```
>> Haversine(1.5)
0.4646313991661485 

>> Haversine(0.5 + 2 * I)
-1.150818666457047+I*0.8694047522371576 
```
