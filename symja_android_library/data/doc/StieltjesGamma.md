## StieltjesGamma

```
StieltjesGamma(a)
```

> returns Stieltjes constant. 
 
See
* [Wikipedia - Stieltjes constants](https://en.wikipedia.org/wiki/Stieltjes_constants)
* [Fungrim - Stieltjes constants](http://fungrim.org/topic/Stieltjes_constants/)

### Examples

```
>> StieltjesGamma(0) 
EulerGamma

>> StieltjesGamma(0,a) 
-PolyGamma(0,a)
```
