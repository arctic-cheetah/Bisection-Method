## Prime

```
Prime(n)
```

> returns the `n`th prime number.
 
See
* [Wikipedia - Prime number](https://en.wikipedia.org/wiki/Prime_number)

### Examples

```
>> Prime(1)
2

>> Prime(167)
991
```
