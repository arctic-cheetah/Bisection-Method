## PreIncrement

```
PreIncrement(x)

++x
```

> increments `x` by `1`, returning the new value of `x`. 

### Examples

`++a` is equivalent to `a = a + 1`:

```
>> a = 2   
>> ++a    
3    
 
>> a    
3    
```
