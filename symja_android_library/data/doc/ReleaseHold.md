## ReleaseHold

```
ReleaseHold(expr)
```

> removes any `Hold`, `HoldForm`, `HoldPattern` or `HoldComplete` head from `expr`.
 

### Examples

```
>> x = 3;

>> Hold(x)
Hold(x)

>> ReleaseHold(Hold(x))
3

>> ReleaseHold(y)
y
```
