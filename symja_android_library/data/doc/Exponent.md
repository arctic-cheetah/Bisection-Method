## Exponent

```
Exponent(polynomial, x)
```

> gives the maximum power with which `x` appears in the expanded form of `polynomial`.
 

See
* [Wikipedia - Exponentiation](https://en.wikipedia.org/wiki/Exponentiation)
 
### Examples

```
>> Exponent(1+x^2+a*x^3, x)
3
```
