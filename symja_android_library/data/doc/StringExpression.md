## StringExpression

```
StringExpression(s_1, s_2, ...)
```

or

```
s_1 ~~ s_2 ~~ ...
```

> represents a sequence of strings and symbolic string objects `s_i`.
  
### Examples

```
>> "a" ~~ "b" ~~ "c" // FullForm
"abc"
```
