## Sinh

```
Sinh(z)
```

> returns the hyperbolic sine of `z`.
 
See
* [Wikipedia - Hyperbolic function](https://en.wikipedia.org/wiki/Hyperbolic_function)
 
### Examples

```
>> Sinh(0)
0
```
