## Degree constant

```
Degree
```

> the constant `Degree` converts angles from degree to `Pi/180` radians.
 
See:
* [Wikipedia - Degree (angle)](http://en.wikipedia.org/wiki/Degree_(angle)) 
 
### Examples

```
>> Sin(30*Degree)
1/2
```

Degree has the value of Pi / 180
```
>> Degree == Pi / 180
True

>> Cos(Degree(x))
Cos(Degree(x))
 
>> N(Degree)    
0.017453292519943295   
```
