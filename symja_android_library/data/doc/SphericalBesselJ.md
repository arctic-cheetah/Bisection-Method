## SphericalBesselJ

```
SphericalBesselJ(n, z) 
```

>  spherical Bessel function `J(n, x)`. 

See
* [Wikipedia - Bessel function - Spherical Bessel function](https://en.wikipedia.org/wiki/Bessel_function#Spherical_Bessel_functions)

### Examples

```
>> SphericalBesselJ(2.5,-5) 
I*0.204488
```
