## SquareFreeQ

```
SquareFreeQ(n)
```

> returns `True` if `n` is a square free integer number or a square free univariate polynomial.

See
* [Wikipedia - Square-free polynomial](https://en.wikipedia.org/wiki/Square-free_polynomial)


### Examples

```
>> SquareFreeQ(9)
False

>> SquareFreeQ(105)
True
```
